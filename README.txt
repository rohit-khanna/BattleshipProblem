Running Sample Inputs:
---------------------
"Input.txt" file is present in the Directory at: "BattleShip_Problem\input.txt". It contains the Sample Scenario shared with Problem statement.

The System will take this file as input.


The test cases can be executed in either ways:
	1. MOdify the Contents of "Input.txt" to have new test cases.
	2. MOdify "App.config" to specify file path in "value" field
		example:   <add key="input" value=""></add>

Note: If the app.config file doesnot have any value for this key , the test cases will be executed from "input.txt" file
